# 🤖 How a Chatbot Works

A chatbot receives a message from the user as input.
It processes the text using Natural Language Processing (NLP) techniques to understand the meaning.
The AI model analyzes patterns and predicts the best possible response.
The system generates a reply based on training data and learned patterns.
Finally, the chatbot sends the response back to the user as output.

---

# 📚 Key Learnings

* AI systems work in stages.
* Input data is processed by AI models.
* Models generate predictions or responses.
* AI products follow pipeline architectures.
* Understanding pipelines helps build better AI systems.
